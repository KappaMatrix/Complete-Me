Complete Me

Complete Me is a JavaScript library for autocompletes. It can suggest results (rather like Google search). And is DOM friendly - a maximum of 10 results are rendered in the DOM at once.

It also doesnâ€™t have any dependencies.

Examples

And some examples of the library can be seen below:


autoComplete2 = new CompleteMe("#auto-complete-2", {
  placeholder: "Type here 2",
  noResultsText: "No results match 2",
  data: data2,
  saveByValue: false,
  selectedKey: "name_contains",
  onSelect: function(value, key) {
    return console.log("onSelect " + key + ": " + value);
  },
  suggestResult: true,
  onSuggestionSelected: function(value) {
    return console.log("onSuggestionSelected " + value);
  }
});
